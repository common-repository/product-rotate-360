== Description ==

= Product Rotate 360 =

You can easily add 360 degree rotating image view to your wordpress site using this simple plugin and short code.

Short code: [product_rotate_360]

Global Settings: Settings -> ProductRotate 360

*   Image Location: URL of your image directory.
*   Image Name Prefix: Prefix in the image name. e.g. shoe-960- for image names (shoe-960-01.jpg, shoe-960-02.jpg etc.)
*   Image Count: Total number of images.
*   Image Name Digits: Number of digits in the image name counter. e.g. 2 for image names (shoe-960-01.jpg, shoe-960-02.jpg etc.)
*   Canvas Width: Width of the canvas in px. e.g. 400 for 400px
*   Canvas Height: Height of the canvas in px. e.g. 400 for 400px
*   Image Extension: Image file extension. e.g. jpg, png etc.
*   Frame Time: To control the rotation speed. any integer > 0, default 100

== Installation ==

Install Product Rotate 360 either via the WordPress.org plugin directory or by uploading the files to your server.

After activating Product Rotate 360, Settings -> ProductRotate 360.

Enter setting values and click save changes.

Now you can add the below shortcode where you want to display the Product Rotate 360 view.

[product_rotate_360]

That's it. You're ready to go!
